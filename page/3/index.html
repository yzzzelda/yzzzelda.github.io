名称 英文 解释
PID控制器 Proportional-Integral-Derivative 比例-积分-微分 广泛应用于控制系统中的反馈控制策略。它通过调整控制输入来使系统的输出达到期望值
PLC可编程逻辑控制器 Programmable Logic Controller 专为工业环境设计的一种数字化计算机控制系统，用于实现工业设备的自动化控制功能
DCS分布式控制系统 Distributed Control System 用于自动化和过程控制的系统，其特征是控制功能分布在整个系统中，而不是集中在一个中央控制室。这种系统通常应用在复杂的工业环境中，如化工厂、发电厂和水处理厂。
RTD电阻温度检测器 Resistance Temperature Detector 用于测量温度的传感器。RTD的工作原理基于金属的电阻会随着温度变化而发生改变。RTD通常使用纯金属（如铂、镍或铜）作为传感元件，其中铂RTD（如PT100、PT1000）是最常用的类型。
RPM Revolutions Per Minute 每分钟转数
SPST Single Pole Single Throw 单刀单掷开关
SPDT Single Pole Double Throw 单刀双掷开关
NO normally open
NC normally closed
COM common 公共端
PPB Parts Per Billion 定义：PPB表示“十亿分之一”，即1 PPB表示在十亿个单位中有一个单位的目标物质。 使用场景：用于检测微量物质的浓度。例如，空气中某污染物的浓度为20 PPB意味着每十亿份空气中有20份该物质。
PPM Parts Per Million 定义：PPM表示“百万分之一”，即1 PPM表示在一百万个单位中有一个单位的目标物质。 使用场景：常用于表示溶液中某成分的浓度。例如，水中10 PPM的污染物意味着每百万份水中有10份污染物。
NPS公称通径（英寸） Nominal Pipe Size NPS和DN通常可以互换使用，但其数值并不总是一一对应。对于NPS和DN之间的换算，通常可参考相应的标准表。例如：
DN公称直径（毫米） Diameter Nominal NPS 1 = DN 25 1英寸=25.4毫米
MLSS混合液悬浮固体 Mixed Liquor Suspended Solids MLSS 指的是活性污泥系统中混合液中的悬浮固体浓度 通常以 mg/L 表示
SS悬浮固体 Suspended Solids SS 是水样中所有不溶于水的固体物质，包含有机和无机悬浮颗粒物 同样以 mg/L 表示
NTU浊度 Nephelometric Turbidity Unit 浊度是一种表示水中悬浮颗粒物对光线散射程度的指标，通常用于衡量水的清澈程度。较高的 NTU 值表示水中的悬浮颗粒较多，水质较浑浊。
ORP Oxidation-ReductionPotential 它表示溶液的氧化还原电位。ORP值是水溶液氧化还原能力的测量指标,其单位是mv
